#### Css选择器类型权重



```
权重计算的规则
内联样式，如：style+“.....”权值为1000
id选择器，如：#content，权值为0100
类，伪类，属性选择器，如content，权值为0010
类型选择器，伪类选择器，如div p，权值为0001
继承的样式没有权值
权重相等的时候，后面的会覆盖前面的

很关键：	
带有关键字！important 的 css 属性 权重无穷大（这个是尚方宝剑，那个敢挡路 就砍谁)
```
